# Dataset Information

## Source
This project uses the **Sample - Superstore Sales Dataset** from Kaggle.

🔗 **Download from:** https://www.kaggle.com/datasets/vivek468/superstore-dataset-final

## How to Use
1. Download `Sample - Superstore.csv` from the link above
2. Place it in this `data/` folder
3. Rename it to `superstore.csv` (or update the filename in the notebook)
4. Run the notebook

## About the Dataset
- **Rows:** ~10,000 sales transactions
- **Columns:** 21 fields including Order Date, Customer, Product, Category, Sales, Profit, Region
- **Time Period:** 2014–2017
- **Business Context:** A US-based retail superstore selling furniture, office supplies, and technology products
